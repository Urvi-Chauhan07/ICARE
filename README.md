ICARE is a role-based healthcare web application that allows Patients, Doctors, and Admins to interact using a secure PHP-MySQL backend.  
It supports user authentication and appointment scheduling with separate dashboards for each role.

 Features

  Patient
- Sign up & login
- Book appointments
- Access patient dashboard

  Doctor
- View scheduled appointments
- Doctor dashboard access

  Admin
- Manage users (patients & doctors)
- Admin panel access

 Tech Stack
- Frontend: HTML, CSS  
- Backend: PHP  
- Database: MySQL
